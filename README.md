# 70015 — Web Toolbox

A lightweight, browser-based toolbox for designers and developers. All processing happens locally; no files or data are uploaded.

## Tools

- **Image Converter** — batch convert images to WebP / AVIF / JPEG / PNG / ICO. Resize, compress, crop, and download as ZIP.
- **Color Palette** — extract dominant colors from images or folders. Copy HEX and export JSON.
- **Base64 Swap** — convert images to Base64 Data URLs, or decode Base64 back to images.

## Usage

Open the site, drop files, done.
